Welcome to the official unofficial repository of guitar transcriptions to many of the songs played by Phish. Actual official transcriptions can be purchased from Phish DryGoods. If you have a correction or new submission, please add or edit the file directly in Github or open an issue. There's also a small selection of old video clips of Trey playing on YouTube.

Bass tabs from Brad Weiger

A trove of over 80 hand-written bass tabs has been generously made available to the community by Brad Weiger, former bassist of the tribute band The Lawn Boys. But we need your help! If you are able and willing to help transcribe these from their present hand-written-and-scanned format to digital, your help will be much appreciated. Please save bass tabs in the bass/ subfolder.

Open-source archive

In an effort to make the information both more freely available to anyone who wants it and easier to maintain, all transcriptions are now simple text files. This information belongs to the community, so you can download a snapshot of the entire archive (tabs, loops, and all) and save it to your hard drive at any time.

How to contribute

To contribute, you'll need to create a Github account, which is free.

Edit a transcription directly

The preferred method is detailed below, but if you can't figure it out, simply open an issue and describe your correction there.

  1. Browse to the file you want to edit (or find it using the search box at the top) and click the "Edit" button near the top of the file.
  2. Make your changes, then click "Propose File Change."
  3. On the next page, click "Send pull request" to send a request to have your changes merged back into the main project. If you've changed your mind and want to delete your changes, click on the "Branches" tab and delete the branch.
  4. Once your request has been accepted, you can delete the old branch (optional).
Create a new transcription
  1. For the sake of simplicity, use the links below:
    • New guitar transcription New bass guitar transcription
  2. Paste or type in the transcription, then click "Propose new file"
  3. On the next page, click "Send pull request" to send a request to have your transcription file merged into the main project.
